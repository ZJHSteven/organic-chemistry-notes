# 杂环化合物
## 目的要求
1. 掌握杂环化合物的概念及命名。
2. 掌握主要的杂环化合物(呋喃、吡咯、噻吩、吡啶、嘧啶、嘌呤)及其衍生物。
3. 了解杂环化合物的**芳香性**。
4. 了解生物碱的结构特征。

## 吡咯、呋喃和噻吩
1. 环内，所有成环原子均为$Sp^2$ 杂化
2. 未参加杂化的p轨道垂直于环面并平行重叠，形成**环状离域大 π键**。
3. 符合 Hückel的 4n+2 规则
4. 稳定性： 苯 ＞ 噻吩＞呋喃＞ 吡咯
5. 亲电取代活性：吡咯＞呋喃＞ 噻吩＞苯
